1. Загальне призначення класу
Клас GameEngine відповідає за логіку основного ігрового циклу, обробку вибору гравця, виклик дій (Action), виведення інформації про стан гри, перевірку завершення та взаємодію з об’єктом Pirate.

2. Пакет та імпорти
package game;
import game.actions.*;
import java.util.Scanner;
import java.util.ArrayList;
import java.util.List;

Файл належить до пакету game. Імпортуються всі класи дій, а також класи для зчитування введення та створення колекцій.

3. Поля класу
private final Pirate pirate;
private final List<Action> actions;
private boolean gameOver;
private int day;

pirate — головний персонаж.
actions — список доступних дій.
gameOver — прапорець, що завершує гру.
day — лічильник днів.

4. Конструктор
Створює пірата зі стартовими ресурсами, ініціалізує список дій, встановлює прапорець завершення гри в false, а день у 1. Потім додає доступні дії HuntAction, SailAction, RestAction, TradeAction.

5. Метод checkGameOver()
Перевіряє, чи пірат живий. Якщо здоров’я або ресурси вичерпано, виводить повідомлення та завершує гру.

6. Метод showDayHeader()
Виводить заголовок дня у зручному форматі.

7. Метод readChoice()
Приймає введення користувача в заданому діапазоні. Використовує Scanner. Перевіряє правильність введення та повертає обране число.

8. Метод playTurn()
Виконує один день гри:
• генерує нові ціни на ринку;
• показує номер дня;
• показує ресурси пірата;
• виводить список дій;
• зчитує вибір гравця;
• виконує вибрану дію;
• перевіряє умови завершення гри;
• збільшує день.

9. Метод showGameStart()
Виводить вступну інформацію та назву гри.

10. Метод showGameEnd()
Виводить завершальне повідомлення та кількість пройдених днів.

11. Метод start()
Головний цикл гри:
• виводить стартове повідомлення;
• запускає цикл днів, у якому викликається playTurn();
• після завершення — показує фінальну статистику.

12. Висновок
GameEngine є центральною логічною частиною гри: керує станом, переходами між днями, вибором дій, обробкою поразки та запуском циклу гри. Структура класу побудована на принципах покрокової текстової RPG.
